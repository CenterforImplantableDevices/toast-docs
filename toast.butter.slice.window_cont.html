toast.butter.slice.window_cont¶

Contains functions applying windowed functions to data.

Before creating new functions, consider exploring the bottleneck library for highly optimized implementations of windowed functions.

toast.butter.slice.window_cont.moving_avg(data, window, axis=- 1)[source]¶

Calculates a moving window average output. Window is cotinuously slid along the data, and is padded to match the shape of the input.

Parameters

  • data (numpy array or ndarray) – Data to process.

  • window (int) – Length of the window (number of points)

  • axis (int, optional) – Array axis to slide the window accross.

Returns

Numpy array matching the shape of the input array, containing the averages along the requested axis.

toast.butter.slice.window_cont.pad_to_length(data, desired_length, axis=- 1)[source]¶

Helper function to pad data on both ends to match a desired length.

The first and last values are used to add values before and after (respectively) the original data to meet the desired length.

Parameters

  • data (numpy array) – The data to be padded

  • desired_length (int) – The length of the output data.

  • axis (int, optional) – Axis where padding should be added.

Returns

Numpy array padded to match the desired length.

toast.butter.slice.window_cont.track_derivatives(data, fs, lpf_cutoff, window, axis=- 1)[source]¶

Calculates the number of first and second derivative zero crossings within a requested window length. The requested window is slid accross the data, and the zero crossings counted.

Data is filtered before derivatives are calculated. A butterworth low pass filter with the requested cutoff is applied. For more detail, see butterworth()

Parameters

  • data (numpy array or ndarray) – Data to process.

  • fs (int) – Sample Frequency of the data.

  • lpf_cutoff (int) – Desired low pass filter frequency cutoff (Hz)

  • window (int) Length of the window (number of points) –

  • axis (int, optional) – Array axis to slide the window accross.

Returns

Two numpy arrays matching the shape of the input array, containing the number of first and second derivative zero crossings along the requested axis.
