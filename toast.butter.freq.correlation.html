toast.butter.freq.correlation¶

toast.butter.freq.correlation.get_stac(data, fs, window, step, mode='full', window_type=None, scaling=1, maintain_signs=True, normalize=False, remove_center=True, exclude_offset=[[]])[source]¶

Slides a window accross the data and calculated the autocorrelation of each window.

Times can be excluded to avoid noise unsing the exclude_times optional arguemnt. To exclude times 20-40 and 200-450: exclude_offset=[[20, 40], [200, 450]]

It will probably be helpful to low-pass filter data before analyzing autocorrelation, to prevent a spike-y output. See the lowpass_filter function lowpass_filter() for more info.

Example of plotting data:

fig, ax = plt.subplots() cs = ax.contourf(stac_time, stac_offsetts, stac_values, cmap=’gnuplot’) _ = fig.colorbar(cs) # to show color scale

Note

Function can only handle one channel at a time.

Parameters

  • data (numpy array) – Array (single row) representing the data to be filtered.

  • fs (int) – Sample rate of the data.

  • window (int) – Size of the window used when computing the periodogram.

  • step (int) – Step size to use when sliding widow across data.

  • mode (str, optional) – Mode to use when calculating autocorrelation. Must correspond to the options in scipy.signal.correlate

  • window_type (str, optional) – Type of window to use for preprocessing. Must be compatible with the scipy get_window function.

  • scaling (int/float, optional) – Exponent to use when scaling the data when converting to power.

  • maintain_signs (bool, optional) – Whether to maintain the sign of the output. If false, all values will be positive.

  • normalize (bool, optional) – Whether to normalize each window independently to range from 0 - 1

  • remove_center (bool, optional) – Will remove the first half of the autocorrelation output. Allows you to see less obvious correlation patterns in the greater offsets.

  • exclude_offset (iterable, optional) – 2D list of time offsets to exclude for plotting. Values in this range are set to the minimum existing coefficient value.

Returns

new_time (numpy array) - Plot X axis. Frequency axis (Hz) corresponding to the output stft coefficients. freq_values (numpy array) - Plot Y axis. of the frequency values corresponding to the output stft coefficients. stac_values (numpy ndarray) - Plot Z axis. Autocorrelation output magnitudes, ready to plot.
