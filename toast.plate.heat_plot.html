toast.plate.heat_plot.train_test_loss(x_axis, training_data, testing_data, accuracy_data=None, training_progress=None, testing_progress=None, x_label='', y_label='Loss', title='', training_label='Train', testing_label='Test', show_plot=False, save_plot=True, save_txt=False, save_path='./', output_folder_name='no_output_folder', description='', bkg_color='w', path_log='./log.txt', close_plot=True, use_agg=True)[source]¶

Plots a typical train-test loss curve to visualize training loss and testing loss over the course of many epochs. Optionally, you can include accuracy data to annotate the best, worst, and last reported accuracy

Parameters

  • x_axis – 1D Iterable: Variable agains which to plot loss. Typically Epochs

  • training_data – 1D Iterable: Training Loss, ordered according to values in x_axis

  • testing_data – 1D Iterable: Testing Loss, ordered according to values in x_axis

  • accuracy_data (iterable, optional) – 1D Iterable: Testing Accuracy, ordered according to values in x_axis

  • training_progress (iterable, optional) – 2D Iterable: 1D-values ordered according x_axis, 2D-value pair representing the min and max training loss, respectively. Meant to summarize training progression

  • testing_progress (iterable, optional) – 2D Iterable: 1D-values ordered according x_axis, 2D-value pair representing the min and max testing loss, respectively. Meant to summarize training progression

  • x_label (str, optional) – use for x axis label

  • y_label (str, optional) – use for y axis label

  • title (str, optional) – use for title

  • training_label (str, optional) – use in legend to identify training_data line

  • testing_label (str, optional) – use in legend to identify testing_data line

  • show_plot (bool, optional) – Whether to render the plot to the screen

  • save_plot (bool, optional) – Whether to save the plot as a .png

  • save_txt (bool, optional) – Whether to save a .txt summary of the final state (min, max, last) of the varaibles for each layer.

  • save_path (str, optional) – Path to save plots to. Note plots will be saved to save_path + output_folder_name location

  • output_folder_name (str, optional) – Folder name to save plots into. Note plots will be saved to save_path + output_folder_name location

  • description (str, optional) – A description of the data in the plot. This will be used as the filename.

  • bkg_color (str -or- tuple, optional) – Character or RBG Thruple representing the background color of the plot.

  • path_log (str, optional) – Folder name to save plots into. Note plots will be saved to save_path + output_folder_name location

  • close_plot (bool, optional) – Whether to close the figure when finished. THis helps minimize memory usage.

  • use_agg (bool, optional) – Whether to use the Agg backend of matplotlib. Use when connecting to non-display machines/servers

Returns

No return value. Generates a plot for the user to either be saved here or viewed later by the user.
