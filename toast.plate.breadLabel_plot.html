toast.plate.breadLabel_plot¶

toast.plate.breadLabel_plot.plot_labels(all_labels_dataframe, save_plots, save_path, filename_header, col_x_axis='datetime', label_key={2: 'before', 3: 'after', 4: 'between'}, plot_skip=[], save_pkl=False, plot_size=[16, 9], plot_dpi=100, plot_linewidth=0.5, plot_close=True)[source]¶

General funcitons used for plotting output to check the resulting labeling data.

Parameters

  • all_labels_dataframe (dataframe) – Dataframe containing all raw data

  • save_plots (bool) – Whether to save the plots

  • save_path (str) – Path to save the data (if save_plots is true)

  • filename_header (str) – Prefix to use for filename to identify the contents of the plot

  • col_x_axis (str, optional) – Column in all_labels_dataframe to use for the x axis.

  • label_key (dict, optional) – Key for the label_ints and the string label. Used in x axis label.

  • plot_skip (list, optional) – List of strings indicating a plot to skip. Options include session, event_startStop, time_beforeEvent, time_afterEvent, time_duringEvent, labels, label_int

  • save_pkl (bool, optional) – Whether to save figure in a pickle file. Note these files can be large files (>1GB).

  • plot_size (list, optional) – Dimensions of each output figure

  • plot_dpi (int, optional) – Pixels per inch to generate when saving the file.

  • plot_linewidth (float, optional) – Linewidth to use for lines in each plot.

  • plot_close (bool, optional) – Whether to close each plot after generation. Recommended to avoid excessive memory use.

Returns

No return value. Creates plots that can be later displayed using plt.show(). Also saves .png files if desired.

General funcitons used for plotting output to check the resulting labeling data.

Parameters

  • all_labels_dataframe (dataframe) – Dataframe containing all raw data

  • segments (list) – List of dictionaries defining segments. See CID convention for this dictionary in toast/bread/README.md

  • sample_rate (int) – Rate that data is sampled at.

  • save_plots (bool) – Whether to save the plots

  • save_path (str) – Path to save the data (if save_plots is true)

  • filename_header (str) – Prefix to use for filename to identify the contents of the plot

  • col_x_axis (str, optional) – Column in all_labels_dataframe to use for the x axis.

  • highlight_border (float, optional) – Number of seconds to leave blank on either side when shading samples. This helps visually separate each sample.

  • highlight_colors (list, optional) – list of colors to use when highlighting each label.

  • save_pkl (bool, optional) – Whether to save figure in a pickle file. Note these files can be large files (>1GB).

  • plot_size (list, optional) – Dimensions of each output figure

  • plot_dpi (int, optional) – Pixels per inch to generate when saving the file.

  • plot_linewidth (float, optional) – Linewidth to use for lines in each plot.

  • plot_close (bool, optional) – Whether to close each plot after generation. Recommended to avoid excessive memory use.

Returns

No return value. Creates plots that can be later displayed using plt.show(). Also saves .png files if desired.
