toast.misc.tlog¶

toast.misc.tlog.output_log(message, file_path='./log.txt')[source]¶

Writes values to a log file and while also printing to stdout. All output messages in toast should use this function to maintain consistency.

This is a convenience function for succinct logging and user update messages. It is best practice to log all progress in a file so that it is available for later reference if questions arise or other users want to inquire about code execution.

Parameters

  • message (str) –

  • file_path (str, optional) – Path to the log file for logging activity. Must include extension, if desired.

Returns

No return value. Message is logged and printed to stdout.
