toast.misc.path¶

Central access source to identify file paths and file locations on different machines.

Machines are monitored based on their MAC address, then mapped to the proper folder location according to what is documented here.

Different functions exist for each dataset to ease syntax.

toast.misc.path.get_filepaths_chbmit(sync_files=False)[source]¶

Returns relevant filepaths for the chbmit dataset according to the machine from which this function was called from.

Machines are identified based on their MAC address.

Parameters

sync_files (bool, optional) – Whether to start a process to synchronize the output path with the CID synology drive. Default is False.

Returns

Path to raw data files of the chbmit dataset. segments_path (str): Path to the sement files of the chbmit dataset. path_output (str): Path to save output files. path_log_file (str): Path to record log files.

Return type

raw_data_path (str)
