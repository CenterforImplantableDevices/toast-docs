toast.butter.filter.notch¶

toast.butter.filter.notch.notch_filter(data, fs, frequencies, aggression, axis=- 1)[source]¶

Creates a theoretically ideal notch filter by places poles and zeros aligned at the target frequency and positioned close to the unit circle, then using those components to calculate filter coefficients. The filter coefficients are then used with filtfilt to filter the passed signal.

By default, the passed data should have the last axis represent the time axis. Additional channels/axes may exist but must be along axes other than the last axis.

The aggression arguments determines how wide-shallow or deep-narrow to make the notch filter. The value should remain between 0 - 1, with closer to 1 indicating a deeper and narrower frequency response. Typical values are anywhere between 0.9 to 0.9995

Parameters

  • data (numpy array) – The data to filter.

  • fs (int) – The sample rate of the data.

  • frequencies (iterable) – Iterable of the frequencies to notch out.

  • aggression (iterable) – Iterable corresponding to the frequencies iterable, defining the aggression, or depth, of the notch filter.

  • axis (int, optional) – The axis which represents the time axis. The filter will be applied along this axis.

Returns

The filtered signal, as a numpy array.
