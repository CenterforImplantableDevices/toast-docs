toast.bread.label.segment¶

toast.bread.label.segment.extract_segments(all_labels_dataframe, sample_rate_expected, sample_length, num_segments, path_log='./log.txt')[source]¶

High level function that helps use labels assigned by label_data to extract metadata that identifies segments within an event.

Parameters

  • all_labels_dataframe (dataframe) – Dataframe containing all raw data and labels

  • sample_rate_expected (int) – Sample rate the data was recorded at (Hz)

  • sample_length (int) – How many data points in a sample. Commonly calculated as seconds*fs

  • num_segments (list) –

    List of the number of segments to extract for each label. There are currently three options for number of segments requested:

    • num_segments > 0 : extract num_segments number of segments

    • num_segments = -1 : extract the maximum number of non-overlapping segments, with the number of segments equal for each label.

    • num_segments = -2 : extract the maximum number of non-overlapping segments as possible for each label.

    • EX: [0, 0, 50, 25] would extract 0 segments for label 0 and 1, 50 segments for label 2, and 25 segments for label 3

  • path_log (str, optional) – Path to the file for outputing logs during execution.

Returns

List of dictionaries identifying the metadata required to identify the requested segments. See CID convention for this dictionary in toast/bread/README.md

toast.bread.label.segment.extract_labelRanges(all_labels_dataframe, sample_rate_expected, labels_to_segment)[source]¶

Use labels assigned by label_data to extract metadata that identifies overall data chunks enclosing an event.

Example of labels_to_segment: If labels_to_segment = [[3], [2,1]], then ranges containing label_int 3 will be identified, and ranges containg either label_int 2 or label_int 1 will be gropued together and identified

Parameters

  • all_labels_dataframe (dataframe) – Dataframe containing all raw data and labels

  • sample_rate_expected (int) – The rate which data is sampled at (Hz).

  • labels_to_segment (list) – List of lists identifying which labels to segment together. Example above.

Returns
